# ゼミ課題１ - 映像分析プラットフォーム

このプロジェクトはゼミ課題として開発された、エンターテイメント特化の映像分析Webアプリケーションです。

## 🎬 プロトタイプ機能

### 実装済み機能
- ✅ **動画アップロード**: ドラッグ＆ドロップ対応
- ✅ **モック分析結果表示**: 
  - 自動ハイライト抽出
  - 感情解析（時系列表示）
  - オブジェクト・人物検出
  - パフォーマンス予測
  - 最適化提案
- ✅ **ダッシュボード**: 統計情報とグラフ表示
- ✅ **レスポンシブUI**: モバイル対応

### 技術スタック

#### フロントエンド
- React 18 + TypeScript
- Tailwind CSS
- Chart.js (データ可視化)
- Lucide React (アイコン)

#### バックエンド
- FastAPI (Python)
- モックAI分析エンジン

## 🚀 デモ実行方法

### 🎯 簡単デモ版（推奨・即座に体験可能）
Node.js不要で即座に体験できます：

```bash
# デモページを開く
映像分析デモを開く.bat
```

または直接ブラウザで開く：
```
demo/index.html
```

**必要な環境：**
- インターネット接続（CDNライブラリ読み込みのため）
- モダンブラウザ（Chrome, Firefox, Edge等）

### ⚙️ フルバージョン（Node.js必要）
完全版を体験したい場合：

1. **Node.js インストール**
   - https://nodejs.org/ からダウンロード・インストール

2. **自動起動**
   ```bash
   start_prototype.bat
   ```

3. **手動起動**
   ```bash
   # フロントエンド
   cd frontend
   npm install
   npm start
   
   # バックエンド
   cd backend
   pip install -r requirements.txt
   python main.py
   ```

### アクセス
- **デモ版**: 直接HTMLファイル（おすすめ）
- **フル版フロントエンド**: http://localhost:3000
- **フル版API**: http://localhost:8000
