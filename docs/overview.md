# プロジェクト概要

## 目的

このプロジェクトはゼミ課題として作成されており、以下の目標があります：

1. Git/GitHubの使用方法の習得
2. プロジェクト管理の実践

- Python 3.x
- Git/GitHub
- Markdown

## 開発環境

- OS: Windows
- エディタ: 任意のテキストエディタ
- バージョン管理: Git
